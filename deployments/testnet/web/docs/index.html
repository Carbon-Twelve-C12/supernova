Quick Start Example

Connect to the Supernova testnet API:

# Get blockchain info curl https://api.testnet.supernovanetwork.xyz/blockchain/info # Get your balance curl https://api.testnet.supernovanetwork.xyz/account/{address}/balance # Submit a transaction curl -X POST https://api.testnet.supernovanetwork.xyz/blockchain/submit \ -H "Content-Type: application/json" \ -d '{"transaction": "..."}'
